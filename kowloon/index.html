九龍六根三合会

kowloon rokkon triad

  • 九龍六根三合会とは、"短BMS"を中心したBMSイベント。
  • 登録期間：2026年6月13日（土）00:00:00(UTC+9) ～ 6月21日（日）23:59:59(UTC+9)。
    インプレ期間：2026年6月22日（月）00:00:00(UTC+9) ～ 7月13日（月）23:59:59(UTC+9)。
  • 当イベントでは三つのカテゴリーが参加可能：
    • <30s：0～30秒の作品。
    • <60s：31～60秒の作品。
    • <90s：61～90秒の作品。
  • 参加作品の「尺」はmBMplayに表示される尺が基準となる。
    • （BMSONの場合では、beatorajaの曲選択画面に表示される尺が基準となる。）
  • 1名の参加は各カテゴリーにつき1作品が登録可能。ただし、
  • 「1名」とは、書いてある名前だけにより成立される。
    • いずれの偽名でも全く別人として扱われます。
    • 正体が明白であろうか誰であろうか何人であろうか問わず。別名＝別人。
  • 1名に登録数超過がある場合、発見した時点からしばらく直されてなければその超過分だけは失格。
    • つまり直してくださるといいね（アーティスト名を偽名に変更）。
  • 以外はbrs003採用とか。

  • this is a "short bms" event.
  • registration: 2026-06-13(Sat) 00:00:00(UTC+9) ~ 06-21(Sat) 23:59:59(UTC+9).
    impressions: 2026-06-22(Sun) 00:00:00(UTC+9) ~ 07-13(Mon) 23:59:59(UTC+9).
  • in this event, there are three enterable "categories":
    • <30s: entries must be 0-30 seconds.
    • <60s: entries must be 31-60 seconds.
    • <90s: entries must be 61-90 seconds.
  • the length of an entry is its length displayed in mBMplay.
    • (in the case of bmson, it is its length displayed during song select in beatoraja.)
  • an artist can register one entry in each category. however,
  • an "artist" is defined solely by the name supplied.
    • any aliases will be treated as completely separate people.
    • the obvious masking, identity or number of persons is not relevant. different name = different person.
  • if excess registrations under one name go unfixed for long enough, the excess will have to be disqualified.
    • i.e. fix it and you'll be golden (change the artist name to an alias).
  • otherwise, brs003 applies (probably).
